Key Insights from Discovery

Integrated Customer View

A unified 360-degree customer profile is critically needed. Data is currently siloed across Adobe Analytics and Salesforce, forcing analysts to consult multiple sources. The primary request is for clean, consolidated data for easier audience segmentation.

Data Cleanliness & Consistency

Data quality is a major hurdle, requiring frequent cleaning. Manual processes and unreliable CRM updates challenge personalization efforts. A universal Party ID is being implemented to address this.

CDP Complexity & Rigidity

The current CDP (ActionIQ) is being replaced. It struggles with complex institutional data and rigid table relationships, making advanced segmentation difficult and requiring heavy IT involvement for changes.

Adobe Analytics Underutilization

The platform is underused due to a lack of dedicated resources. Reporting is cumbersome, data export is challenging, and it fails to capture the full user journey beyond the page level, limiting its strategic value.

Audience Journey Gap

A critical gap exists in connecting web engagement to automated email journeys since Adobe Audience Manager was degraded. The transition to AEP is estimated to take 18-24 months, leaving a functional void.

Technical Debt in AEM

Significant technical debt is accumulating in AEM from pages using old templates. To use new features, pages must be manually rebuilt, requiring a major effort to modernize the web property.

Fully Integrated CDP

With Salesforce Data Cloud replacing ActionIQ, data from Caspian (via Informatica) and Adobe Analytics can be piped directly into the CDP. Data Cloud's native connection to SFMC simplifies ingesting engagement data and pushing segmented audiences, which can automatically trigger journeys. This reduces manual effort. The integration with Adobe Target may require some custom work using Informatica or an SFTP target.

Profile Unification

Data Cloud’s identity resolution can stitch data from Adobe Analytics, SFMC, Caspian, and Salesforce CRM to create unified customer profiles. Matching rules can prevent unclean data from impacting campaigns. This unified view allows for behavior-triggered sends and can bridge the gap left by Audience Manager. Inside Salesforce CRM, unified profiles can be viewed directly from contact/lead records, providing a full 360-degree activity history without needing to check multiple systems.

Robust Analytics

Reporting pain can be eased by using Data Cloud to perform data transformations during ingestion. For advanced business intelligence, it may still be beneficial to push this formatted analytics data back into Caspian for use with a dedicated BI tool. Exporting and manipulating data for specific reporting needs may still require dedicated resources.

Asset Migration

It is strongly recommended that the team migrates all pages using old AEM templates and components to the new standards. Maintaining disparate assets decreases technical stability over time. Pages should be prioritized for rebuilding based on their importance for supporting new, valuable features, such as the CEO search functionality.
